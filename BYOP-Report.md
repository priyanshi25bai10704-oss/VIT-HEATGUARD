# BYOP Project Report - Hydration & Heat Tracker

**Project Title:** VIT-HEATGUARD - Daily Hydration Tracker for Bhopal

**Student Name:** Priyanshi

**Problem Statement**  
Bhopal experiences intense heatwaves every summer because of where it is located on earth. High temperatures lead to dehydration, weakness, fatigue, and heat-related illnesses. Many residents, especially students and outdoor workers, do not drink enough water because they wait until they feel thirsty.

**Why this project matters**  
Proper hydration can prevent early symptoms like weakness and dizziness. This simple tool makes it easy for anyone to know exactly how much water they need based on today’s weather and their personal details.

**Approach & Solution**  
I created a console-based Python program that:
- Takes inputs (temperature, age, activity, weight)
- Calculates recommended water intake using a simple formula
- Provides reminders and warns about dehydration/weakness signs
- Shows remaining water needed for the day

**Key Decisions**  
Kept the project simple using only basic Python concepts (input(), if-elif, arithmetic) so it is easy to understand and run.

**Challenges Faced**  
Making the water calculation realistic while keeping the code short and clear.

**What I Learned**  
- How to apply programming concepts to solve a real local problem in Bhopal.
- Importance of user-friendly output and health awareness during summer.
- Simple code can have real-world impact.
